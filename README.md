SuperMart Inventory

SuperMart Inventory is a lightweight Swift-based supermarket inventory management system. It allows you to add products, track stock levels, update quantities, and persist data locally using JSON. Perfect for small stores or learning Swift file I/O and Codable.

Features:
✅ Add, update, and list products
✅ Track stock levels and highlight low-stock items
✅ Persist inventory locally using JSON (inventory.json)
✅ Filter and manage products easily
✅ Extensible for categories, sorting, and total inventory value
